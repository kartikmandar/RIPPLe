# RIPPLe Documentation

This directory holds the project documentation for **RIPPLe** (Rubin Image
Preparation and Processing Lensing engine). Hand-written user guides, API
reference, and tutorials should be added here.

## Layout

- *(future)* user guide, API reference, tutorials — the actual RIPPLe docs.
- `_build/`, `generated/` — generated Sphinx output (gitignored).
- `superpowers/` — local design specs and implementation plans created during
  development (gitignored; kept on disk for reference, not version-controlled).
